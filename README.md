# KEMET-Formula-AI-hackathon
Formula AI hackathon 2022 for weather forecasting

The project is the work of the Kemet team participating in Hackmakers Formula AI Hackathon. Specifically, the Data Analytics theme.

"Branched_Model.hdf5" : A pre-trained model that got the best results, it can be loaded with keras

"Branched_Network.ipynb" : The Notebook containing our work

"Branched_Network.pdf" : A PDF format for the notebook for visual inspection

"Clean_Filtered_Data" : A csv file carrying the data after cleaning and preprocessing

"Profiling Reports" : 2 reports explaining the data before and after the cleaning process

"weather.zip" : A zip file carrying the raw data used
